# 🐾 Pet Grooming Landing Page

A **professional, fun & artistic** landing page built for pet grooming stores. Designed to impress customers and showcase grooming services with style.

## ✨ Features
- Hero section with a bold CTA
- Services showcase with icons
- Testimonials section
- Gallery grid
- Contact / Booking section
- Fully responsive design
- Smooth scroll & animations

## 🚀 Getting Started
Just open `index.html` in your browser — no build step required!

## 🛠 Tech Stack
- HTML5
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript
